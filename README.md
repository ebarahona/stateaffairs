# StateAffairs Exercise

**React Native: React Native Expo and React Navigation Project**

## Prereqs

- **yarn** is preferred
- **Expo** npx expo ...

---

## Installation

Run one of the following commands to install dependencies:

```sh
yarn install
# or
npm install
```

## Run

```sh
npx expo start
```
